The Linux Quizz

Le 27 Décembre 2020

The Linux Quizz est un test de 12 questions portant sur le système d'exploitation Linux et ses nombreuses applications.

Le but de ce quizz est avant tout de manipuler une base de données MySQL à partir d'un code PHP, permettant une authentification par mire de login, la gestion d'une session et l'incrémentation du score d'un joueur. Ce quizz permet de donc de :

  • Créer un compte en ligne (les identifiants sont enregistrés dans la base de données (BDD))
  • Se connecter avec un nom d'utilisateur, une adresse email valide et un mot de passe
  • Jouer au quizz avec des variables de session éphémère.
  • Enregistrement des résultats dans la base de données.
  • Déconnexion

Nous remercions alors qcmquiz.com et quizz.biz pour leurs questions pertinentes portant sur GNU Linux.

Hébergement

Pour l'hébergement du quizz en Php, il est nécesssaire de posséder un environnement LAMP, car PHP est un langage orienté serveur.
J'ai donc procédé à la crétation d'une machine virtuelle KVM (Linux) puis à l'installation d'un environnement LAMP dessus (Linux Apache MySQL Php).


Pour ce qui reste de la publication, j'ai recréé l'environnement LAMP de ma machine virtuelle sur une machine hébergée sur AWS (Amazon Web Service). En théorie, l'installation de Lamp est désormais très simple, il suffit d'entrer :

$ sudo apt-get install lamp-server^

Malgré tout, plusieurs problèmes émergent lors de l'installation d'un certificat SSL/TLS, ou de phpmyadmin par exemple, pour qu'il soit configuré AVEC apache et ait accès aux fichiers de configuration de mysql. De plus, la plateforme AWS a ses propres spéciictés telles que le type de machine (Amazon Linux 2, Amazon AMI, ...) et le Firewall, la clé de chiffrement pour la connexion au serveur via SSH, donc il a fallu s'adapter pour pouvoir installer cet environnement de travail et avoir accès aux fichiers par le moyen de l'adresse public et d'un navigateur.

Enfin, près une installation lente, il suffit d'uploader l'ensemble de mes fichiers avec sftp, ainsi que la base de données par drag and drop sur phpmyadmin.

Arborescence

L'arborescence de mes fichiers se présente comme suit :

  • /
    • index.php (... page d’accueil ...)
  • jeu/ (... partie du site accessible uniquement par les membres ...)
    • quizz.php (... page de lancement du jeu avec les niveaux ...)
    • quest.php (... page avec une question ...)
    • gagne.php (... page lorsque le joueur a gagné ...)
    • score.php (... page contenant les meilleurs scores ...)
  • php/
    • bd.inc.php (... paramétrage de la BD ...)
    • connexion.inc.php (... fichier de connexion à la BD ...)
    • mire.inc.php (... mire de login ...)
    • authorise.inc.php (... l’utilisateur est connecté ...)
  • css/
    • quizz.css (... le fichier CSS ...)
  • img/ ( ... contient l’ensemble des images ...)
    • tux.png

Base de Données

La base de données du quizz doit répondre à plusieurs besoins :

  • Authentifier les utilisateurs
  • Héberger la liste des questions
  • Héberger les différentes réponses, et annoter la bonne
  • Compter le score d'un ou plusieurs joueurs

Pour répondre à ces besions, la BDD se structure ainsi :
(les * représentent les clés primaires)

Joueur Questions Score
Login [ VARCHAR 32 ] [ PRIMARY ] Identifiant [ VARCHAR 32 ] [ PRIMARY ] Identifiant [ VARCHAR 32 ] [ PRIMARY ]
Mot de passe [ VARCHAR 32 ] Enoncé (question) [ VARCHAR 256 ] Date [ TIMESTAMP ]
Email[ VARCHAR 64 ] Réponse 1 [ VARCHAR 32 ] Joueur [ VARCHAR 32 ] [ PRIMARY ]
- Réponse 2 [ VARCHAR 32 ] Score [ INT ]
- Réponse 3 [ VARCHAR 32 ] -
- Réponse 4 [ VARCHAR 32 ] -
- N° bonne réponse [ INT ] -
- Difficulté (1, 2 ou 3) [ INT ] -
