Linux测验

2020年12月27日

Linux Quizz是有关Linux操作系统及其许多功能的12个问题测验 应用程序。

此测验的目标首先是从PHP代码处理MySQL数据库，从而允许 通过登录模式进行身份验证，管理会话并增加玩家的分数。 测验 因此允许：

  • 创建一个在线帐户（标识符保存在数据库（BDD）中）
  • 使用用户名，有效的电子邮件地址和密码登录
  • 使用临时会话变量进行测验。
  • 将结果记录在数据库中。
  • 登出

因此，我们要感谢qcmquiz.com和quizz.biz与GNU Linux有关的问题。

住宿

要在Php中托管测验，必须有一个LAMP环境，因为PHP是一种语言 面向服务器。
因此，我着手创建KVM虚拟机（Linux），然后进行安装 一个LAMP环境（Linux Apache MySQL Php）。


对于剩下的内容，我在计算机上重新创建了虚拟机的LAMP环境。 托管在AWS（Amazon Web Service）上。 从理论上讲，安装Lamp现在非常容易，只需输入：

$ sudo apt-get install lamp-server^

尽管如此，安装SSL / TLS证书或phpmyadmin时仍会出现一些问题， 这样就可以使用apache对其进行配置，并且可以访问mysql配置文件。 此外，AWS平台具有其自身的特殊性，例如机器类型（Amazon Linux 2，Amazon AMI等）和防火墙，用于通过SSH连接到服务器的加密密钥， 因此，我们必须进行调整以能够安装此工作环境，并能够通过公共地址和浏览器访问文件。

最后，安装缓慢后，您要做的就是上传我所有的 带有sftp的文件，以及通过在phpmyadmin上拖放的数据库。

数据库

测验数据库必须满足几个需求：

  • 验证用户
  • 托管问题列表
  • 托管不同的答案，并注释正确的答案
  • 计算一个或多个玩家的分数
