更改分区管理器（RAID => AHCI）后，我毫不奇怪地意识到Windows 10不会 不再展示， 计算机在启动时显示黑屏，然后在没有磁盘的情况下启动到另一个磁盘上 找到Windows 10工作。

-工具

这是KVM的赌注 (Kernel-based Virtual Machine)


KVM是一种集成到linux中的开放源代码虚拟化技术，它允许， 简单地说，从内核本身启动虚拟机。 这种技术的作用是使虚拟机非常强大，因为它们使用 与 Oracle相比，系统资源要好得多 例如VirtualBox 。


我经常使用KVM来启动虚拟机，让我测试a 环境灯，MySQL数据库服务器，甚至使用Kali Linux为其 笔测试工具。

所以我想尝试使用KVM的Windows 10，并在手上的我的管理 CPU资源（GNOME扩展）

令我惊讶的是，Windows 10流动旋转。 我显然促进了安装 轻型程序，但即使在缩放工作中播放视频。


当然，拥有一个适度的显卡（英特尔UHD图形），这种流动性会很快进入 银行面向使用美食游戏或4 k流媒体。 但这些表演结果 足以我的利用。

- 功能和安装

要创建KVM虚拟机，我使用 virt-manager GUI工具，但一切都可以完成 在线命令感谢 virsh 和 qemu 。

在线订单，它会给：

# virt-install --name=win10 \
--vcpus=4 \
--memory=16024 \
--cdrom=/tmp/windows10.iso \
--disk size=100 \
--os-variant=windows10

-平滑

这家公司的理想将能够通过按下我的按钮来启动Windows 10 码头（我最喜欢的酒吧） 这正是我所做的。

步骤1 ：

创建启动虚拟机的脚本，并启动桌面查看器。

start_windows10.sh
#! /bin/bash

virsh start win10
virt-viewer -cf qemu:///system win10

然后授予必要的授权：

chmod +x ./start_windows.sh

第2步 ：

在应用程序菜单中创建一个图标

再一次，有不同的方法可以在应用程序菜单中创建和编辑图标。

我可以在~/.local/share/applications 目录中创建一个.desktop文件。但是 我刚刚使用了软件 alacarte 在图形模式中轻松编辑。
